# 🐺 WagerWolf - Crypto Odds Hunter

**Hunt the best crypto odds across blockchain sportsbooks!**

WagerWolf is an aggressive Telegram bot that tracks, analyzes, and hunts the most profitable crypto betting opportunities across multiple platforms.

## 🎯 Features

- 🐺 **Crypto Odds Aggregation**: Best odds in USDT, BTC, ETH, SOL
- 🔄 **Arbitrage Detection**: Risk-free profit opportunities
- 🎯 **Sniper Alerts**: Real-time odds movement notifications
- 💰 **Value Bet Identification**: High-confidence crypto picks
- 💳 **Wallet Integration**: Track your crypto betting
- ⚡ **Premium Alpha Wolf**: Exclusive features for serious hunters

## 🚀 Quick Start

1. Clone: `git clone https://github.com/YOUR_USERNAME/WagerWolf.git`
2. Install: `pip install -r requirements.txt`
3. Configure: Create `.env` with API keys
4. Hunt: `python bot.py`

## 📋 Commands

- `/hunt` - Start hunting odds
- `/odds <league>` - Get crypto odds
- `/arbitrage` - Find arbitrage opportunities
- `/sniper` - Set sniper alerts
- `/wallet` - Manage crypto wallet
- `/premium` - Unlock Alpha Wolf features
- `/help` - Wolf's guidance
